視差捲動效果

作者：Vineet Kumar

什麼是視差捲動？

視差捲動是一種網頁設計技巧，讓網站的背景移動速度慢於前景。當訪客向下捲動頁面時，便會 產生類似 3D 的效果，增加深度感並創造更沉浸的瀏覽體驗。視差的原理源自光學錯覺。由於 人眼會將距離較近的物體視為比遠處的物體更大，我們也會覺得遙遠的物體移動得較慢。這種 錯覺長久以來被運用在各種媒介中的視差效果上，營造逼真的呈現。最早可以追溯到傳統動畫， 例如迪士尼的《白雪公主與七個小矮人》，以及《超級瑪利歐》等電玩遊戲。隨著 CSS 與 HTML 的進步，視差效果也逐漸發展成為今日網頁設計的一部分。

最早在瀏覽器中實作視差捲動的案例之一，由網頁開發者 Glutnix 於 2007 年在部落格分享， 提供了範例程式碼與示範，使用 JavaScript 與 CSS 2，並支援當時的 Internet Explorer 6 等 瀏覽器。不過直到 2011 年 HTML5 與 CSS3 問世後，這項技術才真正受到網頁設計師的青睞。 支持者認為這是擁抱網路流動性的一種簡易方法。他們以視差背景作為工具，更有效吸引 使用者並提升網站整體體驗。然而，普渡大學在 2013 年發表的研究指出：「……雖然視差捲動 增強了使用者體驗的某些面向，但未必能全面提升整體體驗。」


為什麼要使用視差圖像？

    設計師應該嘗試視差圖像的兩個主要理由：

  • 有助於說故事

    視差圖像非常適合用來說故事。良好的設計在於溝通——當我們設計網站時，希望與訪客 交流，講述關於產品、服務或品牌的動人故事。視差捲動讓我們能為訪客打造獨特的線上 體驗。甚至可以建立單頁式網站，讓訪客不需切換頁面就能讀完整個品牌故事。

  • 有助於提升使用者互動

    視差圖像可用來提升使用者互動。降低網站的跳出率是網頁設計中值得重視的目標之一。 跳出率衡量造訪者只瀏覽單一頁面後便按下「返回」或關閉分頁的比例。高跳出率會影響 搜尋排名，因此任何能降低跳出率的作法都很值得嘗試。精心打造的視差頁面能帶給使用 者動態體驗，讓他們停留更久。


若想在網站加入視差捲動效果，請留意以下建議。

  • 選擇性地使用

    將視差捲動運用在特定元素上會更有力量，而不是在所有頁面與時間都使用。可將它放在 頁首、標題，或僅用於首頁。視差捲動應該強化網站設計，而非分散使用者注意力。

  • 圖像壓縮

    視差捲動會搭配各式媒體檔案與 CSS 形狀來創造深度。別讓過大的圖片檔拖慢載入速度， 進而犧牲網站效能與使用體驗。

  • 善用色彩

    視差捲動不僅與前景和背景的圖像有關，也與色彩息息相關。可以在背景使用較為低飽和的 色調，前景則運用更鮮明的顏色，以營造更強烈的深度感。
